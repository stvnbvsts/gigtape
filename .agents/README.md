# Agent Workspace

This folder contains agent-neutral project guidance and reusable skills.

Use this as the source of truth for shared AI assistant material. Tool-specific
folders such as `.claude/`, `.codex/`, or `.cursor/` should only contain local
settings, compatibility notes, or adapters required by that tool.

## Layout

- `skills/`: reusable procedural skills, each with its own `SKILL.md`.
